# PrayogShala

A multilingual AI-powered coding gym that transforms data structures and algorithms concepts into guided real-world projects. Learn practical DSA through project-based learning, then prove your understanding with AI-powered evaluation.

**Vision:** Learn → Build → Prove

---

## Problem

Students often memorize DSA concepts but struggle to apply them in real software development or practical projects. Existing platforms focus on isolated coding questions rather than practical skill verification. There's a significant gap between theoretical knowledge and the ability to implement solutions in real-world scenarios.

## Solution

PrayogShala combines multilingual AI tutoring, project generation, automated code execution, and AI-driven viva assessments to help learners build and prove their skills. It transforms the learning journey into:

1. **Learn** - Understand concepts with Sarvam AI in multiple languages
2. **Build** - Create real projects generated by Gemini based on learned concepts
3. **Prove** - Demonstrate understanding through AI-powered code-aware vivas

## Core Features

- **Multilingual Tutor** - Sarvam AI provides learning content in multiple languages
- **Concept Validation** - Test understanding before moving to projects
- **Guided Practice** - Structured problem-solving with hints and guidance
- **Project Generation** - Gemini generates real-world projects based on learned topics
- **Milestone Learning** - Break projects into manageable milestones
- **Monaco Editor** - Professional code editing environment
- **Judge0 Execution** - Automated code testing and execution
- **AI Code Review** - Sarvam provides detailed code analysis
- **AI Code-Aware Viva** - Personalized viva questions based on actual implementation
- **Skill Reports** - Verified skill documentation with proof

## User Workflow

1. **Language Selection** - Choose preferred language for learning
2. **Topic Selection** - Pick from available DSA modules and topics
3. **Concept Validation** - Test understanding of core concepts
4. **Sarvam Tutor** - Learn with AI-powered multilingual tutoring
5. **Practice Question** - Solve guided practice problems
6. **Project Unlock** - Unlock real-world project after concept mastery
7. **Gemini Project Generation** - Receive AI-generated project with milestones
8. **Monaco Editor** - Write code in professional editor with syntax highlighting
9. **Judge0 Execution** - Run and test code against test cases
10. **AI Code Review** - Get detailed feedback on implementation
11. **AI Code-Aware Viva** - Answer viva questions based on your code
12. **Skill Report** - Receive verified skill report upon completion

## AI Responsibilities

**Sarvam AI:**
- Teaching and concept explanation

- Code review and feedback
- Viva evaluation
- Skill report generation

**Gemini API:**

## System Architecture

```
React Frontend 
    ↓
Express Backend 
    ↓
MongoDB Atlas 
    ↓
Sarvam AI + Gemini API 
    ↓
Judge0 (Code Execution)
```

**Data Flow:**
- Frontend handles UI and user interactions
- Backend manages authentication, data storage, and API orchestration
- MongoDB stores user progress, submissions, and skill reports
- AI services handle content generation and evaluation
- Judge0 provides secure code execution and testing

## What Makes PrayogShala Different

Most platforms generate projects or provide AI tutors — PrayogShala does both, and then **proves understanding** through a code-aware AI viva. The viva analyzes the student's own implementation to ask targeted questions about algorithm choices, data structures, optimizations, and failed test cases. Generic answers don't pass. Real understanding does.
